Flex Code - don't forget the semicolon!


  • flex-direction
    • row or row-reverse
    • column or column-reverse
  • justify-content for horizontal spacing
    • 'vertical' if flex-direction is column
    • center
    • flex-start or flex-end
    • space-around or space-between
  • align-items is vertical spacing
    • 'horizontal' if flex-direction is column
    • center
    • flex-start or flex-end
